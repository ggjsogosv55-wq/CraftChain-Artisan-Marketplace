# CraftChain Artisan Marketplace

## Overview

CraftChain Artisan Marketplace is a blockchain-based platform designed to verify handmade craft authenticity, support local artisans, and prevent mass-produced counterfeits from being misrepresented as authentic handmade goods. Built on the Stacks blockchain using Clarity smart contracts, this system empowers traditional craftspeople while protecting cultural heritage.

## Mission

Our mission is to create a transparent, decentralized marketplace that:
- Authenticates genuine handmade crafts through blockchain verification
- Supports local artisans with fair compensation and recognition
- Preserves traditional craft techniques and cultural heritage
- Prevents exploitation through mass production counterfeiting
- Builds consumer trust in authentic handmade goods

## System Architecture

### Smart Contracts

The CraftChain ecosystem consists of four core smart contracts:

#### 1. Artisan Profile Registry (`artisan-profile-registry.clar`)
- Verifies artisan credentials and traditional techniques
- Maintains cultural heritage craft backgrounds
- Creates immutable profiles for authenticated craftspeople
- Tracks artisan reputation and community standing

#### 2. Handmade Verification System (`handmade-verification-system.clar`)
- Records photo and video proof of handcraft processes
- Implements time-stamped creation documentation
- Validates authentic production methods
- Ensures transparency in the creation process

#### 3. Cultural Heritage Protection (`cultural-heritage-protection.clar`)
- Protects traditional craft techniques from appropriation
- Prevents unauthorized mass production of cultural designs
- Maintains registry of protected traditional methods
- Enforces cultural authenticity standards

#### 4. Artisan Support Rewards (`artisan-support-rewards.clar`)
- Distributes token incentives for purchasing authentic goods
- Rewards consumers who support local craftspeople
- Implements loyalty programs for sustainable craft consumption
- Creates economic incentives for the handmade ecosystem

## Technical Implementation

### Blockchain Platform
- **Network**: Stacks Blockchain
- **Language**: Clarity Smart Contracts
- **Consensus**: Proof of Transfer (PoX)
- **Security**: Bitcoin-grade security inheritance

### Data Storage
- **On-chain**: Critical authenticity data, artisan profiles, verification records
- **Off-chain**: Media files (photos/videos) stored on IPFS with on-chain hashes
- **Hybrid**: Optimized for cost efficiency while maintaining security

### Token Economics
- **Utility Token**: Rewards authentic purchases and artisan participation
- **Staking Mechanism**: Artisan reputation and consumer loyalty programs
- **Governance**: Community voting on cultural protection policies
- **Sustainability**: Fee structure supports ongoing platform development

## Getting Started

### For Developers

1. **Prerequisites**
   - Node.js 16+ installed
   - Clarinet CLI tool
   - Git for version control

2. **Installation**
   ```bash
   git clone https://github.com/ggjsogosv55-wq/CraftChain-Artisan-Marketplace.git
   cd CraftChain-Artisan-Marketplace
   npm install
   ```

3. **Development**
   ```bash
   clarinet check          # Validate contract syntax
   clarinet test           # Run test suite
   clarinet console        # Interactive testing
   clarinet deploy         # Deploy to testnet
   ```
